# FIRE OS 2.2.1

Mobile Net Worth UI Edition.

## 更新重點

- 參考行動端 Net Worth / 資產管理 App 的深綠毛玻璃風格
- 手機版改為底部 Tab Bar，接近原生 App 操作感
- Dashboard、Assets、Liabilities、Settings 視覺密度提升
- 保留 FIRE OS 2.2 的資料管理能力：Assets / Liabilities / Investments / Cashflow / Journal / Firestore sync / PWA

## Deploy

Upload all files in this folder to your GitHub repository root:

- index.html
- manifest.json
- service-worker.js
- assets/
- README.md
