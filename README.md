# FIRE OS 2.1

新增內容：

- PWA 支援：可在手機或桌面安裝成 App。
- Service Worker：快取核心檔案，提升載入速度，基本離線可開啟。
- Firebase Google Login + Firestore 雲端同步維持不變。
- Dashboard、Assets、Investments、Cashflow、FIRE、Journal、AI Coach 維持 2.0 功能。
